# 🌐 Network Sniffer using Python

## 📌 Overview
This project demonstrates how to capture and analyze network traffic using Python.  
A network sniffer intercepts packets flowing through a network and displays useful information such as source IP, destination IP, and protocol.

---

## 🎯 Objectives
- Capture live network packets
- Identify source and destination IP addresses
- Detect protocols (TCP, UDP)
- Understand basic network communication

---

## 🛠️ Tools & Technologies
- Python 3
- Scapy library
- Npcap (required for Windows)

---

## ⚙️ Installation

### 1. Install Python
Download from: https://www.python.org/

# 🔐 Keylogger Simulation (Safe & Educational)

## 📌 Overview
This project is a **safe simulation of a keylogger** built using Python.  
It demonstrates how keystroke logging works in a controlled environment.

⚠️ This program does NOT capture system-wide keystrokes.  
It only logs input entered into the application.

---

## 🎯 Objectives
- Understand how keylogging works
- Simulate keystroke logging safely
- Store input into a file
- Analyze cybersecurity risks

---
